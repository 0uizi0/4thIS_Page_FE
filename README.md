## Convention

---

### Git

| Tag      | Desc.             |
| -------- | ----------------- |
| 문서     | 문서 수정         |
| 소스수정 | 기능 및 버그 수정 |
| 소스추가 | 새로운 기능 추가  |
| 파일추가 | 새로운 파일 추가  |

### Code

**CSS**

- CSS 방법론 중 BEM 방식을 사용한다.
- reset.css로 CSS 초기화를 한다.
- components와 screens로 나누고 style.css에는 등록해 관리한다.

**Naming**

- 변수명은 카멜 표기법을 따른다.
- 상수는 대문자로 스크림 스네이크 표기법을 따른다.
- 배열은 복수형 단어를 사용한다.
